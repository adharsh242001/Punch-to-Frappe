Manual Date Range Push

The server cannot directly reach most edge PCs. Run this generated command on PC-A or PC-B to fetch a custom range and upload it to this server.

How Remote Control Works

Current design is push-only: edge PCs connect to the server and upload events. The server does not have a live shell or direct access to the PCs.

Can server fetch from edge PC directly?No, not with the current architecture.
Can server run commands on PC-A/PC-B?No. That would require a separate authenticated command channel.
Safe future optionEdge agent polls the server for limited jobs like fetch range, show health, restart agent, or upload diagnostics.
Unsafe optionGeneral remote shell commands. Avoid this unless you add strong auth, auditing, allowlists, and encryption.
